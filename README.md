# DBQuery

A modern, fast database client for <!-- engines:shipped-and -->PostgreSQL, MySQL, MariaDB, SQLite, SQL Server, MongoDB, CockroachDB, Percona, TiDB, and Azure SQL<!-- /engines:shipped-and -->.

## About

DBQuery is a native desktop application built with Tauri, React, and Rust. It provides a powerful interface for managing your databases with features like:

- AI-powered Text-to-SQL query generation
- Visual table editing with inline cell updates
- SSH tunnel support for secure remote connections
- Data import/export (CSV, JSON, XLSX, SQL)
- Schema diff and migration tools
- Real-time performance monitoring
- Multiple editor groups with split view
