# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, email **security@dbquery.ai** with:

- a description of the issue and its impact,
- steps to reproduce (or a proof of concept), and
- the affected component (desktop app, website, or licensing API) and version.

We aim to acknowledge reports within 3 business days and will keep you updated on
remediation progress. Please give us a reasonable window to release a fix before any
public disclosure.

## Scope

DBQuery is a local-first desktop database client. Your database credentials and queries
stay on your machine; the hosted services handle only licensing, accounts, payments, and
the optional Cloud AI proxy. Reports against any of these are welcome.

## Supported versions

Security fixes target the latest released version of the desktop app and the currently
deployed hosted services.
