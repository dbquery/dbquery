# Contributing to DBQuery

Thanks for your interest in DBQuery! This repository is for **issue tracking and community
discussion only** — it does not contain the application source code, so we don't accept code
pull requests here.

## How you can help

- **Report a bug** — [open a bug report](../../issues/new?template=bug_report.yml). Include your
  OS + DBQuery version, the database engine, and clear steps to reproduce.
- **Request a feature** — [open a feature request](../../issues/new?template=feature_request.yml)
  describing the problem you're trying to solve and your use case.
- **Ask or discuss** — use [Discussions](../../discussions) for questions, ideas, and feedback.

Before filing, please [search existing issues](../../issues) to avoid duplicates and add a 👍 to
ones that matter to you — it helps us prioritize.

## Security

Found a vulnerability? Please follow our [Security Policy](./SECURITY.md) and email
**security@dbquery.ai** instead of opening a public issue.
